# Kodi Backup Plus

An enhanced version of the Kodi backup addon with additional features for comprehensive system backup.

## Features

- Backup and restore your complete Kodi configuration
- Enhanced backup options including:
  - Root config.txt file backup
  - Skins backup
  - User addons and addon data
  - Database files
  - Playlists
  - Thumbnails and fanart
  - Profiles
  - Game saves
  - All configuration files

## Installation

1. Download the latest release
2. Install through Kodi's addon manager using "Install from zip file"
3. Configure backup settings in the addon settings menu

## Backup Options

### Simple Mode
- **Root Config File**: Backs up the main config.txt from Kodi root directory
- **Skins**: Backs up all installed skin addons
- **User Addons**: Backs up installed addons
- **Addon Data**: Backs up addon settings and data
- **Database**: Backs up Kodi's databases
- **Playlists**: Backs up custom playlists
- **Thumbnails/Fanart**: Backs up cached images
- **Config Files**: Backs up all configuration files
- **Profiles**: Backs up user profiles
- **Game Saves**: Backs up game save data

### Advanced Mode
- Custom file and folder selection
- Advanced backup set configuration
- Fine-grained control over what gets backed up

## Scheduling

- Daily, weekly, or monthly automated backups
- Custom schedule support
- Option to run missed backups on startup
- Configurable backup rotation

## Storage Options

- Local storage
- Network storage (SMB/NFS)
- Dropbox support
- Compressed backup support (ZIP format)

## Requirements

- Kodi 19 or higher
- Sufficient storage space for backups
- Network connection for cloud storage options
